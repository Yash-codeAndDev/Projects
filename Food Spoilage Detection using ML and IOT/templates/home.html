Help

This is a food quality checking system, merging Internet of Things (IoT) and machine learning technologies. Leveraging machine learning and IoT, we aim to provide an effective low-cost, e-nose that can predict the expiry of fruits so that they can be consumed accordingly.

How to Use?

Simply put the temperature, humidity, and gas evolution values manually or via sensors.

What is the relation between gas released and fruit expiry?

As the fruit spoils, it releases various gases. The MQ2 sensor can detect those gases, which can be used to predict the expiry.

What is the use of temperature and humidity then?

Environment conditions like temperature and humidity impact the amount of gas released from the fruit.

From the Developers...

The project is still in its initial state and is currently only able to predict the expiry of bananas. As the data set will grow, we'll be able to predict the expiry of a wider array of fruits :/
